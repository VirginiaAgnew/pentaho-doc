In instances when you want to work on your model and may not have access to the Pentaho Server, you can save your security information in a file. The Pentaho Metadata Editor will retrieve your settings from the file instead of accessing the server every time you open your domain.

  1. Follow the instructions in Define Security Settings .
  2. After you click Test, all of the XML between the tags, including content the tags themselves, is provided in the XML dialog box.
  3. Copy then paste this XML code into your favorite text editor, and save the file as metadata_security.xml  in a location of your choice.
  4. Click OK to close the dialog box.
  5. Click the File tab in the Security Service dialog box.
  6. Browse to the file that you just saved.
  7. Click OK to exit the dialog box.
