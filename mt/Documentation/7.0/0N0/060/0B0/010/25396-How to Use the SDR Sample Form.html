Overview

The Streamlined Data Refinery (SDR) is made up of a few different tools which work in conjunction to make the data refining process easier for the user. Data is refined in PDI with the Build Model job entry and the Annotate Stream step, then pushed to Business Analytics through the Publish Model job entry in the Pentaho Server. From there, report designers can use the SDR form to cherry-pick their data for  Analyzer reports. These instructions walk you through how to use the sample SDR form created by Pentaho for example purposes.

The Movie Ratings - SDR Sample Form

The Movie Ratings-SDR Sample form is easy to use. All you will need to do is select the check boxes next to the filters which you want to apply and the SDR will refine the data for your Analyzer report.

We recommend that you make sure to do at least some selecting of data for optimum results.

  1. Log on to PUC using a logon with administrator permissions.
  2. Open the Movie Ratings-SDR Sample form.

The All Requests Processed field contains a list of up to ten instances of data set activity. You can click an item in the list to work with that data set or create a new one.

  1. Click to select the Genre filters which you want to use from the Movie Review Filter panel.
  2. Click to select profile items in the Member Profile panel to narrow down your data to specific points.

  1. Enter the Start Date for the data set. Optionally, use the date picker to choose the start date.
  2. Enter the End Date for the data set. Optionally, use the date picker to choose the end date.
  3. Indicate whether you would like to Grant Access to the model to Everyone or keep it for only your use.
  4. Enter a data source name in the field.
  5. Click the Let's do this button to run the data set and launch a new Analyzer report.

  1. Depending on whether you keep the check box selected, Analyzer launches with your data set parameters in place. Optionally, you can click Go to Analyzer when you are ready to begin creating reports.
