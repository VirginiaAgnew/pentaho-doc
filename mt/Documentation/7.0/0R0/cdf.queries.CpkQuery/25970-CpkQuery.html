cdf .queries . CpkQuery

 
Static

Class that represents a CPK query. This class will be registered globally using the static dashboard function registerGlobalQuery .

The constructor of this class is created dynamically and registered in the dashboard query container queryFactories .

To create a new CPK query use the dashboard function getQuery .

Source: queries/CpkQuery.js, line 24

Example

dashboard.addDataSource("myCpkQuery", {queryType: "cpk", ...});
dashboard.getQuery({dataSource: "myCpkQuery"}).doQuery(successCallback, errorCallback);
