This guide comes with a set of sample web pages which demonstrate the ways to integrate Pentaho Server functionality and content into your web application. It also describes the techniques used by the various samples, so when following this guide, we recommend you deploy the samples on your Pentaho Server.

Audience: Developers with HTML and JavaScript experience. 

Prerequisites: This guide assumes you have a correctly installed and configured the Pentaho Server running locally on your machine. It also assumes you know how to start and stop the server if necessary. Please refer to the following sections for help with installing, configuring and administering the Pentaho Server:

  • Install Pentaho Software
  • Configure Pentaho Systems
  • Fine-Tune Pentaho Systems

Locating the Samples

The samples are packaged into a file named pentaho-sdk.zip accessible for download in your Customer Support Portal . The readme file contains the information for deployment.

Deploying the Samples

To deploy the samples to your Pentaho Server, follow these steps:

  1. Stop the Penthao Server.
  2. Run SDK > Installer.bat (or the corresponding script for your OS) to uncompress these two folders to your selected location: the integration-examples folder and platform-plugins .
  3. Copy the integration-examples directory to your pentaho-server/tomcat/webapps directory.
  4. Open the platform-plugins directory and copy these two folders: example-visualization and oem-tools .
  5. Paste them into the .../pentaho-server/pentaho-solutions/system directory.
  6. Restart the Pentaho Server.

The samples are deployed and can be accessed by opening a web browser and navigating to http://localhost:8080/integration-examples .

If you want to access the OEM tools, browse to http://localhost:8080/pentaho/conten...web/index.html .

If you want to access the example-visualization, look under the Charts menu in Analyzer. It will be listed as Example KPI .
