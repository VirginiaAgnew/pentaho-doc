To deliver content as well as expose various services, we use a REST web interface. These sections comment on the various samples and how they use the REST URLs.

For the full reference of the Pentaho REST API, please refer to http://javadoc.pentaho.com .

General Notes

The REST APIs often allow interaction with repository files. The REST API expects references to repository files to be specified using the : character as file separator.
