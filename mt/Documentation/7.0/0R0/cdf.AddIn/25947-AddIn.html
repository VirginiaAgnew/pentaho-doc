cdf . AddIn

Creates a new add-in. Add-ins come in two varieties: Static add-ins represent static data or behavior, while Scriptable add-ins represent dynamic, context-dependent behavior.

Methods Details

call (target, state, options) : Object | Object

Call the add-in. If the add-in is static, all parametersare irrelevant, and this method will simply return the valueof _value.

In a dynamic add-in, the implementation will be passed tothe target DOM Element (whatever element is relevant, e.g. the element that was clicked on or the table cellthat is being processed), a state object with whatevercontext is relevant for the add-in to fulfill its purpose,and optionally any overriding options.

Components are allowed to pass undefined as the target if no elements make sense in context.
