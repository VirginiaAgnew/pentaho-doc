Upload Specific Arguments

Command Description Values Required
-i,--import Upload Command n/a Yes
-x,--source External system type legacy-db or file-system (default) Yes
-o , --overwrite Overwrites file(s) on upload. Default value is True. Boolean No
-m, --permission Applies ACL using manifest file. Default value is True. Boolean No
-r, --retainOwnership Replaces the file ownership upon upload with the ownership of the original download. Default value is True. Boolean No
-t, --type The type of content being uploaded - files (default), metadata.  File type No

Command Description Values Required
-e, --export Download command n/a Yes
-fp,--filepath Location that the zip file is downloaded to File path Yes
-w, --withManifest If true, includes Manifest.xml inside zip. If false, download excludes this file. Boolean No
