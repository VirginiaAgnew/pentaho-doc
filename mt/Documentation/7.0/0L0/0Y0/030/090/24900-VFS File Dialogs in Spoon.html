Some job and transformation steps have virtual filesystem (VFS) dialog in place of the traditional local filesystem windows. VFS file dialogs enable you to specify a VFS URL in lieu of a typical local path. The following PDI and PDI plugin steps have such dialogs:

  • File Exists
  • Mapping (sub-transformation)
  • ETL Meta Injection
  • Hadoop Copy Files
  • Hadoop File Input
  • Hadoop File Output
Note: VFS dialogs are configured through certain transformation parameters. Refer to Configure SFTP VFS for more information on configuring options for SFTP.
