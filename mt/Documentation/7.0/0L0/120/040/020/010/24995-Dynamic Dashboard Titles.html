While in Dashboard, each panel has the ability to show a title with current parameters.

  1. Click the edit (pencil) icon in the top toolbar. The Edit Pane appears at the bottom of the screen
  2. Click the report that has the parameters you are going to change. You will need to go through this process for each report to which you want to add parameters.
  3. Click the Add Parameter button. Placed parameter appears after the title.
  4. Click Apply
  5. Close the Edit Pane by clicking the edit (pencil) icon.

The title will now show the parameters applied to it.
