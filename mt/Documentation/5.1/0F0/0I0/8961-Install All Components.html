Overview

The Wizard is the easiest and quickest way to install the Pentaho Business Analytics (BA) Suite. The BA Suite consists of BA and Data Integration (DI) components. After installation, you can use the software to create reports and visualize data, create and schedule data integration processes, model data, and perform other processing on traditional and big data sources. The Wizard Installation process consists of preparing your environment, installing the software, and next steps for after installation.
