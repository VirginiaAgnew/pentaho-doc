There are only a few last things you need to do, and then you are done configuring the BA Server and User Console.

  1. Document how you have configured the BA Server. When you get a newer version of Pentaho, this helps you decide which components need upgrading and how to configure the upgraded components if necessary.
  2. Share the BA Server URL with team members who want to create web-based data analysis, reports, and dashboards. This enables them to login to the User Console and get started with the Web-Based Analysis, Reports, and Dashboards Tutorial .
  3. If you want to use the BA design tools, please go on to configure them now . This way they will be ready to use when you need them.
