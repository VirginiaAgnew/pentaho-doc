Overview

Provides a step-by-step guide for configuring the BA design Tools.

Before you can use the BA design tools, you need to perform configuration tasks for each workstation that runs the design tools. Just follow the Guide Post graphics to ensure you complete the entire process. It should take you less than 1 hour for each workstation running the design tools.

If you used the Pentaho Trial Download and the Installation Wizard, the BA Server and its design tools reside on the same workstation. If you used any of the other installation techniques, the BA Server and design tools may reside on separate workstations. Here is the list of the design tools and what they do.

Design Tool What It Does
Aggregation Designer Optimizes the multidimensional Mondrian data model
Metadata Editor Creates relational data models and refines the models created with the Data Access Wizard
Report Designer Creates print-quality reports
Schema Workbench Creates multidimensional Mondrian data models and refines the models created with the Data Access Wizard
