To edit an existing Instaview data source, from Configure mode, within the Configure data source panel, click Edit . The data source input dialog appears. Or, if a template accepts parameters constraints there will be controls or fields that allow you to select the parameters on which to filter your data.

When configuring the input settings for a data source, the Auto run Analysis when ready option, if checked, will automatically create a new View after pressing Run or Refresh .
