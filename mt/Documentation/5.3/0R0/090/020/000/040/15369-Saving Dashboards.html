You must be in Edit mode to save a dashboard.
  1. Click the Save As button, which is a floppy disk and pencil button, to open the Save As dialog box.
  2. In the File Name text box, type a file name for your dashboard.
  3. Enter the path to the location where you want to save the dashboard. Alternatively, use the up/down arrows or click Browse to locate the solution (content files) directory in which you will save your dashboard.
  4. Click Save . The report saves with the name specified.
