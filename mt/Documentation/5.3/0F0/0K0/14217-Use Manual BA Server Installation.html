Overview

This guide explains how to manually install the BA Server on Tomcat or JBoss.  If you already have Tomcat or JBoss installed and you want to deploy BA on it, use these instructions.  If you are fine with installing Pentaho on its own instance of a Tomcat server, or if you want to evaluate the product, we highly recommend that you use the installation wizard to Install All Components instead.
