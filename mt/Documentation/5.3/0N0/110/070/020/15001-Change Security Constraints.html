Follow the instructions below to change security constraints on a specific business table or column.

  1. Open the Properties dialog box that contains the table or column you want to change.
  2. Click (Add Property). The Add New Property dialog box appears.
  3. Select the Security Information property and click OK . The Properties Editor (Settings) scrolls down to the Metadata Security section.
  4. Add the individual role or user permissions to the business model, table, or column. These permissions be enforced in the Pentaho BA Platform after publishing the new metadata model.
