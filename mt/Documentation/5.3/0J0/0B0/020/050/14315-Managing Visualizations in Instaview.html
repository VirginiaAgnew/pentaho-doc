From the Visualization panel you can

  • Select and view previously saved Views
  • Delete visualizations created in Instaview
  • Rename a visualization
  • Create new Analyzer Reports and visualizations from the existing Instaview

To create a new Analyzer report from an exisiting Instaaview, click the New Analysis button, which brings you back to --check-- View mode with a blank Analyzer canvas. You can open and edit existing Instaview visualizations by selecting a file from the list found in the Visualization panel.
