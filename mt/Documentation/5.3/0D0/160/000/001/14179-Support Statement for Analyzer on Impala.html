Using Analyzer on Impala

These are the minimum requirements for Analyzer to work with Impala.

  1. Use minimum Pentaho BA Suite EE 5.1 or later.
  2. Use minimum Cloudera CDH5.x, Impala 1.3.x or later.
  3. Recommend using Parquet compressed file format for tables in Impala.
  4. Use Apache Hive JDBC that was distributed as part of the CDH shim.
  5. Make sure that the JDBC driver is dropped into the BA Server and Schema Workbench directories.
  6. Turn off connection pooling in Pentaho BA Server.
  7. Set global order by limit in Cloudera manager.
  8. In Mondrian schemas, divide dimension tables with high cardinality into several levels.
