Overview

Explains what to do after the DI Migration.

After you have verified the migration and upgraded Spoon, perform post-migration tasks. These include completing the following tasks.

  • Configure the DI Server .
  • To learn more about how to administer the DI Server, read Administer DI Server .
  • Deprovision the old server and provision the new one in accordance with your organization's procedures.
  • Review Getting Started with DI to learn how to use Spoon.
