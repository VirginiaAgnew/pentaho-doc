Visualizations, such as graphs, charts, maps, and grids, graphically display data to convey a message. Instaview offers many ways to visually display data using the --check-- Analyzer canvas . To create a visualization in Instaview you first need to create a data table by placing measures and dimensions within the Available fields panel on to the --check-- Analyzer canvas . Once your table has the information you want to display, select a chart from the Chart type menu by pressing the Chart type drop-down arrow on the Analyzer toolbar.
