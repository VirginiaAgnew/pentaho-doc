Overview

Explains what to do after you have installed PDI.

Congratulations! You have just installed PDI. Now, you are ready to do these things.

  • See Set up DI Server and Tools to learn how to start the DI Server and Spoon, and to see the instructions on how to install licenses.
  • Work through the Get Started with DI tutorial.

Note: If you have installed the DI Server and you want to migrate content from the previous version of the servers to this one, make sure that your license keys have been installed, then review the DI Upgrade instructions.
