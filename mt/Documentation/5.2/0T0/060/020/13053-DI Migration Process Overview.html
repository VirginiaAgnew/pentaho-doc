Overview

Provides an overview of the DI Migration process.

The migration process consists of the steps indicated in the guidepost . Each step consists of one or more tasks.

  • Prepare Migration Plan : Prepare your migration strategy.
  • Prepare Environment : Install the 5.x.x version of the server, download and unpack the Migration Tool.
  • Export Data From 4.4.x Server : Use the Migration Tool to export data from the 4.4.x server to migration bundle file.
  • Import Data To 5.x.x Server : Use the Migration Tool to import migration bundle file to the 5.x.x server. Manually migrate additional data, as needed.
  • Verify migration : Use checklists to test that migration was completed properly.
  • Next Steps : Indicates what to do after the migration process is complete.
