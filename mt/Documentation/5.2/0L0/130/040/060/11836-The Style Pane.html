The Style pane displays all of the visual and positional style options for any given item in the Structure pane. Click on any Structure element, and the composition of the Style pane will adjust to show all of the available style properties, listed by group.

Note: You cannot edit any Style or Attributes options for any selected report elements in the workspace while the Data tab has focus. Click the Structure tab to see the Style and Attributes panes for selected elements.
