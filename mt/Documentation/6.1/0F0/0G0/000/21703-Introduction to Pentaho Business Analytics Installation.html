Overview

Provides an introduction to the Wizard installation instructions.

The Wizard is the easiest and quickest way to install the Pentaho Business Analytics (BA) Suite. The BA Suite consists of BA and Data Integration (DI) components. After installation, you can use the software to create reports and visualize data, create and schedule data integration processes, model data, and perform other processing on traditional and big data sources.

We recommend using this installation method for evaluation purposes only.

Expertise

These instructions are written for anyone who wants to install the Pentaho BA Suite, or individual BA and DI components.

Tools

You must supply a server or workstation that meets the hardware and operating system requirements indicated in support technologies .

Login Credentials

All of the tasks in these instructions require that you have the computer privileges or accesses you need to install software on your server or workstation.

Installation Methods

Before you start, choose whether you want to install using the default or custom methods. The default installation installs the entire Pentaho BA suite, while the custom installation allows you to choose which pieces that you want to install.
