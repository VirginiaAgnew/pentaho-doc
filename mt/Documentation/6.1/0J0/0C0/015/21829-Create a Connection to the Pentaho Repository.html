Overview

Shows you how to create a connection to the DI Repository.

Transformations, jobs, and schedules are stored in the Pentaho Repository.  Create a connection to the DI Server if you want to access these things, or use the DI Operations Mart audit feature.

  1. Make sure the DI Server is running.
  2. Start Spoon.
  3. Select Tools > Repository > Connect to open the Repository Connection window .

​

  1. If a connection to the Pentaho Repository has not already been created:
    1. Click the Add button.  The Select the repository type window appears.
    2. Select Pentaho Repository (Recommended)  and click OK .  
    3. In the Repository Configuration  window, enter a  Name and Description  for your repository.  
    4. Modify the URL associated with your repository if necessary, then c lick the Test  button to ensure that the URL is properly configured.  If the test fails, make sure that the DI Server is up and the port number in the URL is correct.  (If you installed the DI using the Wizard, the correct port should appear in the installation-summary.txt  file.  The file is in the root directory where you installed PDI.)
    5. Click  OK  to close the  Success  dialog box.
    6. Click OK to close the Repository Configuration window. Your new connection appears in the list of available repositories.
  2. To connect to the repository,  select the repository, then enter the admin (or non-admin) username and password for the repository.
  3. Click OK .
